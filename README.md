# Community Tech App

## Getting started

1. Clone this repo

```bash
git clone https://conzmr@bitbucket.org/conzmr/community-tech-app.git
```

2. Change into the directory

```bash
cd community-tech-app
```

3 Install the dependencies

```bash
yarn
# or
npm install
```

4. Run the project

```bash
react-native run-ios
# or
react-native run-android
```
